normal_plots module¶

normal_plots.C_normal_distrib_neg(wafer_id, sessions, structures, dies)¶

Used to plot the normal distribution of negatives values of C inside a wafer, following selected filters. We first get data inside the database and then plot it in a figure. We use a probability scale, and plot the reference line and the 90% confidence interval :param wafer_id: ID of the wafer :param sessions: Selected sessions :param structures: Selected structures :param dies: Selected dies :return: The plot, converted into base64
